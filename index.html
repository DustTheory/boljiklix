BoljiKlix

Userscript plugin that adds various features to the klix.ba website. It's an unoficial plugin that has no affiliation with the website.

Features:

  • Notification support - uprates, downrates and comments
  • Thread watcher
  • Collapsible comment chains
  • User tag highlighting
  • Improved navigability and readability of comments
  • Admin deleted comments alerts

Check out the project on github

Install

Firefox

Install Violentmonkey, Tampermonkey or Greasemonkey. Next, click here to install the script.

Chrome

First install Violentmonkey or Tampermonkey. Next, click here to install the script.

Safari

Install Tampermonkey, then click here to install the script.

Microsoft Edge

Install Tampermonkey, then click here to install the script.

Other browsers

BoljiKlix should work on any browser that can run userscript. This includes a lot of chromium based browsers, and some minimal WebKitGTK+ / QtWebKit / QtWebEngine browsers. After installing a userscript manager you can here to install the script.
